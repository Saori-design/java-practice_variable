Javascript Function & Scope Practice Lab

Complete the task found below each block of instructions (written as comments) in the Javascript panel of this page.

Setup

To get started, create a project folder and add the file index.html, with the necessary document template. Create a files named script.js and paste the instructions found in the JS file (panel to the right) into it. Attach that <script> to index.html and begin.

Instructions

For each exercise, follow the instructions to create a function that either outputs the result to the console using console.log directly in the function, or has a return with the necessary value and is then output to the console from where the function was called.

Ideally, try both methods to get comfortalbe with the role of a function.
